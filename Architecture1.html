Static Website Hosting

Host a static website in S3 with an serverless architecture. June 05, 2021

Here i have hosted a Personal website in AWS cloud, using an Serverless architecture. The use of Serverless architecture has grown enormously when compared with the serverfull architecture. Main reasons are Cost, easy to scale application and higher performance rate.
